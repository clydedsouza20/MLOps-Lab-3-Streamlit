# MLOps Lab: Streamlit Deployment
**Author:** Clyde Dsouza 
**Institution:** Northeastern University  

## Project Overview
This lab demonstrates the deployment of a Machine Learning model using Streamlit. Unlike the base repository, this project implements:
- **Custom Dataset:** Utilizes the Wine Quality dataset instead of Iris.
- **Model Change:** Implements a Random Forest Regressor for continuous quality scoring.
- **UI Enhancements:** Custom CSS styling and data visualization components.

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Launch app: `streamlit run app.py`
